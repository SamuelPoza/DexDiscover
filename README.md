# DexDiscover
Esto consistiría en una aplicación en la cual puedes introducir el nombre o número de un Pokémon, y esta te dará todos los datos sobre él. En caso de que no conozcas el nombre, también podrás pulsar sobre una imagen 
para que la app te proporcione información, como por ejemplo, las habilidades, estadísticas, tipos, etc. Además de eso, quizás también tenga una opción para ordenarlos dependiendo de lo que necesites. Todo esto con el
objetivo de agilizar la obtención de información acerca de los Pokémon. La aplicación, en principio, obtendría la información de internet y la mostraría por pantalla mediante una interfaz personalizada
